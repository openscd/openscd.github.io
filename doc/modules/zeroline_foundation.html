  • startMove<E, P>(editor: E, Child: new () => E, Parent: new () => P): void
    • Defined in src/zeroline/foundation.ts:174

    The move action can be aborted by clicking on something other than a Child or Parent editor or by hitting the escape key on the keyboard.
